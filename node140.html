Java Map

The Map interface of the Java collections framework provides the functionality of the map data structure. It implements the Collection interface. In Java, elements of Map are stored in key/value pairs. Keys are unique values associated with individual Values. A map cannot contain duplicate keys. And, each key is associated with a single value. We can access and modify values using the keys associated with them.These classes are defined in the collections framework and implement the Map interface.
